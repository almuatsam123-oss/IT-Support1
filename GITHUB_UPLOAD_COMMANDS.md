# GitHub Upload Commands

Run these commands from Git Bash, PowerShell, or Command Prompt after installing Git.

```bash
cd "C:\Users\User\Documents\Codex\2026-06-11\create-a-simple-it-asset-management"
```

```bash
git init
git branch -M main
git remote add origin https://github.com/almuatsam123-oss/It-Amjaad.git
git status
git add .
git commit -m "Initial commit - Amjaad IT Asset Management System"
git push -u origin main
```

```bash

git push -u origin main
```
